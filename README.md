# ZAF BAT Execution OS V6

Architecture statique GitHub-ready, pensée comme base produit sérieuse.

## Structure
- `index.html` — shell principal
- `app.js` — logique produit V6
- `styles/app.css` — styles application
- `styles/zafbat-docs.css` — styles documentaires
- `templates/` — templates HTML documents séparés du code
- `app-helpers/zafbat-docs.js` — renderer documentaire

## Ce que V6 apporte
- colonne vertébrale de données plus propre
- pipeline projet plus explicite
- coût MO planifié / réel / variance / marge estimée
- base partenaires approfondie
- matching chantier pondéré
- documents générés depuis les données projet
- snapshots documentaires figés
- quick field mode
- paiements / incidents / reviews reliés aux projets

## Limite honnête
Cette version reste une application statique front-only. Pour en faire une vraie base vendable à plusieurs utilisateurs :
- authentification réelle
- base distante
- rôles / permissions
- stockage serveur
- logs serveur
- workflows de signature
