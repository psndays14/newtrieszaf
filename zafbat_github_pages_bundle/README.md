# ZAF BAT App (GitHub Pages)

## Déploiement
1. Mets `index.html` à la racine du repo (pas `Index.html`).
2. GitHub → **Settings → Pages**
   - **Deploy from a branch**
   - Branch: `main` (ou `master`)
   - Folder: `/ (root)`
3. Ouvre: `https://<username>.github.io/<repo>/`

## Sauvegarde des données
- Les données sont stockées dans le navigateur (localStorage).
- Utilise les boutons **Exporter / Importer** sur le tableau de bord pour faire des backups JSON.

## Notes importantes
- Le “mot de passe” dans ce fichier est un verrou UI côté navigateur, pas une sécurité réelle.
